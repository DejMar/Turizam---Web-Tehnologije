Objašnjenje koda

Konstante za konekciju

define('DB_HOST', 'localhost');
define('DB_NAME', 'turizam_sobe');
define('DB_USER', 'root');
define('DB_PASS', '');
define() kreira konstante koje se ne mogu mijenjati tokom izvršavanja. Svi fajlovi koriste iste podatke za pristup MySQL bazi.

Funkcija getDB() — konekcija na bazu

function getDB(): PDO {
    static $pdo = null;           // čuva konekciju između poziva
    if ($pdo === null) {
        $dsn = 'mysql:host=' . DB_HOST . ';dbname=' . DB_NAME . '...';
        $pdo = new PDO($dsn, DB_USER, DB_PASS, [
            PDO::ATTR_ERRMODE => PDO::ERRMODE_EXCEPTION,
            PDO::ATTR_DEFAULT_FETCH_MODE => PDO::FETCH_ASSOC,
        ]);
    }
    return $pdo;
}
static $pdo — singleton pattern: konekcija se kreira samo jednom. PDO::ERRMODE_EXCEPTION baca grešku ako SQL ne uspije. FETCH_ASSOC vraća rezultate kao asocijativni niz (npr. $row['naziv']).

Funkcija e() — zaštita od XSS

function e(string $value): string {
    return htmlspecialchars($value, ENT_QUOTES, 'UTF-8');
}
Pretvara specijalne HTML znakove (<, >, ") u sigurne entitete. Koristi se pri ispisu korisničkih podataka: <?= e($soba['naziv']) ?> — sprečava ubacivanje zlonamjernog JavaScripta.

Flash poruke

function flash(string $type, string $message): void {
    $_SESSION['flash'] = ['type' => $type, 'message' => $message];
}

function getFlash(): ?array {
    if (!isset($_SESSION['flash'])) return null;
    $flash = $_SESSION['flash'];
    unset($_SESSION['flash']);    // briše nakon čitanja — prikaže se samo jednom
    return $flash;
}
Flash poruka se sprema u sesiju prije redirect(), a prikazuje na sljedećoj stranici u header.php. unset() osigurava da se poruka ne prikaže ponovo pri osvježavanju.

Objašnjenje koda

Pokretanje sesije

session_start();
require_once __DIR__ . '/../config/database.php';
session_start() mora biti prije bilo kakvog HTML outputa. Omogućava čuvanje $_SESSION podataka (npr. ID prijavljenog korisnika) između učitavanja stranica.

Provjera prijave i uloge

function isLoggedIn(): bool {
    return isset($_SESSION['korisnik_id']);
}

function requireAdmin(): void {
    requireLogin();
    if (!isAdmin()) {
        flash('error', 'Nemate dozvolu...');
        redirect('index.php');
    }
}
isset() provjerava da li je korisnik prijavljen. requireAdmin() prvo poziva requireLogin(), pa tek onda provjerava ulogu — dvostruka zaštita admin stranica.

Funkcija login()

function login(string $email, string $password): bool {
    $stmt = getDB()->prepare('SELECT * FROM korisnici WHERE email = ? AND aktivan = 1');
    $stmt->execute([$email]);
    $user = $stmt->fetch();

    if ($user && password_verify($password, $user['lozinka'])) {
        $_SESSION['korisnik_id'] = $user['id'];
        $_SESSION['uloga'] = $user['uloga'];
        return true;
    }
    return false;
}
prepare() + execute([$email]) — prepared statement štiti od SQL injection. password_verify() upoređuje unesenu lozinku sa hašom iz baze. Uspjeh → ID korisnika ide u sesiju.

Provjera dostupnosti sobe

function sobaJeDostupna(int $sobaId, string $datumOd, string $datumDo): bool {
    $sql = "SELECT COUNT(*) FROM rezervacije
            WHERE soba_id = ?
            AND status IN ('na_cekanju', 'potvrdena')
            AND datum_od < ? AND datum_do > ?";
    $stmt->execute([$sobaId, $datumDo, $datumOd]);
    return (int) $stmt->fetchColumn() === 0;
}
Traži preklapajuće rezervacije. Uslov datum_od < datumDo AND datum_do > datumOd detektuje bilo koje preklapanje perioda. Vraća true ako je COUNT = 0 (soba slobodna).

Izračun cijene

function izracunajCijenu(float $cijenaPoNoci, string $datumOd, string $datumDo): float {
    $od = new DateTime($datumOd);
    $do = new DateTime($datumDo);
    $nocenja = max(1, (int) $od->diff($do)->days);
    return $nocenja * $cijenaPoNoci;
}
DateTime::diff() računa razliku između datuma. max(1, ...) osigurava minimum 1 noć. Rezultat: broj noćenja × cijena po noći.

Baza: SELECT iz tabele sobe (samo dostupna = 1)

Objašnjenje koda

Čitanje filtera iz URL-a

$tip = $_GET['tip'] ?? '';
$pretraga = trim($_GET['pretraga'] ?? '');
$page = max(1, (int) ($_GET['page'] ?? 1));
$_GET sadrži parametre iz URL-a (npr. ?tip=apartman&page=2). ?? '' daje prazan string ako parametar ne postoji. max(1, ...) sprečava negativne stranice.

Dinamički SQL upit

$where = 'WHERE dostupna = 1';
$params = [];

if ($tip && in_array($tip, ['jednokrevetna', 'dvokrevetna', ...])) {
    $where .= ' AND tip = ?';
    $params[] = $tip;
}

$countStmt = getDB()->prepare("SELECT COUNT(*) FROM sobe $where");
$countStmt->execute($params);
$totalSoba = (int) $countStmt->fetchColumn();

$sql = "SELECT * FROM sobe $where ORDER BY cijena_po_noci ASC LIMIT 6 OFFSET $offset";
$stmt->execute($params);
in_array() validira tip — samo dozvoljene vrijednosti idu u SQL. ? placeholderi + $params štite od SQL injection. LIMIT/OFFSET su cijeli brojevi iz koda (ne iz korisničkog unosa). fetchColumn() vraća jedan broj (COUNT).

Prikaz soba u HTML-u (petlja)

<?php foreach ($sobe as $soba): ?>
    <h2><?= e($soba['naziv']) ?></h2>
    <a href="rezervacija.php?soba_id=<?= (int) $soba['id'] ?>">Rezerviši</a>
<?php endforeach; ?>
foreach prolazi kroz niz soba iz baze. <?= ... ?> je kratki echo. (int) cast osigurava da je ID broj u URL-u. e() štiti naziv sobe od XSS.

Objašnjenje koda

Zaštita stranice

$sobaId = (int) ($_GET['soba_id'] ?? $_POST['soba_id'] ?? 0);
Rezervacija zahtijeva prijavu. ID sobe se čita i iz GET (otvaranje forme) i iz POST (slanje forme) pomoću lančanog ??.

Validacija i INSERT

if ($_SERVER['REQUEST_METHOD'] === 'POST') {
    $datumOd = $_POST['datum_od'] ?? '';
    // ... validacija datuma, kapaciteta ...

    if (empty($errors) && !sobaJeDostupna($sobaId, $datumOd, $datumDo)) {
        $errors[] = 'Soba nije dostupna u izabranom periodu.';
    }

    if (empty($errors)) {
        $ukupnaCijena = izracunajCijenu(...);
        $stmt = getDB()->prepare('INSERT INTO rezervacije (...) VALUES (?, ?, ...)');
        $stmt->execute([$_SESSION['korisnik_id'], $sobaId, ...]);
        flash('success', 'Rezervacija je uspješno poslata.');
        redirect('moje-rezervacije.php');
    }
}
$_SERVER['REQUEST_METHOD'] === 'POST' razlikuje slanje forme od običnog otvaranja stranice. Greške se skupljaju u niz $errors. $_SESSION['korisnik_id'] povezuje rezervaciju sa prijavljenim korisnikom. Status ostaje default na_cekanju iz baze.

admin/izvjestaji.php

Admin

Svrha: Mjesečni izvještaj o zauzetosti soba i prihodu.

GET parametri

ParametarOpis
mjesecFormat YYYY-MM (default: tekući mjesec)

Šta računa

  • Ukupna zauzetost — procenat zauzetih noćenja u mjesecu
  • Zauzetost po sobi — prolazi kroz svaki dan mjeseca i provjerava rezervacije
  • Prihod — SUM cijena potvrđenih i završenih rezervacija
  • Statusi — broj rezervacija po statusu u mjesecu

Računaju se samo rezervacije sa statusom potvrdena i zavrsena (za zauzetost i prihod).

Baza: SELECT iz sobe i rezervacije (više upita u petlji)

Objašnjenje koda

Određivanje perioda

$mjesec = $_GET['mjesec'] ?? date('Y-m');
$prviDan = "$mjesec-01";
$posljednjiDan = date('Y-m-t', strtotime($prviDan));
$brojDana = (int) date('t', strtotime($prviDan));
date('Y-m-t') — t daje broj dana u mjesecu (28–31). Za 2026-06 → posljednji dan je 2026-06-30.

Računanje zauzetosti po danima

for ($dan = 1; $dan <= $brojDana; $dan++) {
    $trenutniDatum = sprintf('%s-%02d', $mjesec, $dan);
    foreach ($rezervacije as $rez) {
        if ($trenutniDatum >= $rez['datum_od'] && $trenutniDatum < $rez['datum_do']) {
            $zauzetiDani++;
            break;
        }
    }
}
Dvostruka petlja: za svaki dan u mjesecu provjerava sve rezervacije te sobe. sprintf('%02d') formatira dan sa vodećom nulom (01, 02...). break — jedan dan se broji samo jednom čak i ako ima više rezervacija.

Prihod u mjesecu

SELECT COALESCE(SUM(ukupna_cijena), 0) FROM rezervacije
WHERE status IN ('potvrdena', 'zavrsena')
AND datum_od >= ? AND datum_od <= ?
COALESCE(..., 0) vraća 0 ako nema rezervacija (SUM bi inače bio NULL). Filtrira po datumu dolaska u izabranom mjesecu.

$sql = file_get_contents(__DIR__ . '/sql/schema.sql');
$statements = array_filter(array_map('trim', explode(';', $sql)));

foreach ($statements as $statement) {
