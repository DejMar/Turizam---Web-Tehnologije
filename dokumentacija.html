1. Pregled aplikacije

Aplikacija je web sistem za upravljanje hotelskim sobama i online rezervacijama. Korisnici mogu pregledati sobe, kreirati rezervacije i pratiti njihov status. Administratori imaju pristup panelu za upravljanje rezervacijama, sobama, korisnicima i izvještajima o zauzetosti.

Tehnologije

  • HTML — struktura stranica
  • CSS — stilizacija i responzivni dizajn
  • JavaScript — validacija formi i dinamički izračun cijene
  • PHP — serverska logika, autentifikacija, rad sa bazom
  • MySQL — pohrana podataka (korisnici, sobe, rezervacije)

Glavne funkcionalnosti

  • Pregled i filtriranje soba
  • Online rezervacija sa provjerom dostupnosti
  • Registracija i prijava korisnika
  • Administracija rezervacija, soba i korisnika
  • Izvještaji o zauzetosti po sobama i mjesecima

4. PHP backend — objašnjenje koda

config/database.php

Centralna konfiguracija aplikacije. Sadrži:

  • getDB() — kreira PDO konekciju na MySQL (singleton pattern — jedna konekcija po zahtjevu)
  • e() — escape HTML karaktera radi zaštite od XSS napada
  • formatDate(), formatPrice() — formatiranje datuma i cijena za prikaz
  • flash() / getFlash() — poruke korisniku nakon preusmjeravanja (npr. "Uspješna prijava")
  • redirect() — preusmjeravanje na drugu stranicu
function getDB(): PDO {
    static $pdo = null;
    if ($pdo === null) {
        $pdo = new PDO($dsn, DB_USER, DB_PASS, [...]);
    }
    return $pdo;
}

includes/auth.php

Upravlja autentifikacijom i autorizacijom korisnika:

  • session_start() — pokreće PHP sesiju za čuvanje prijave
  • isLoggedIn() — provjerava da li postoji $_SESSION['korisnik_id']
  • currentUser() — učitava podatke trenutnog korisnika iz baze
  • isAdmin() — provjerava da li korisnik ima ulogu admin
  • requireLogin() — zaštita stranica koje zahtijevaju prijavu
  • requireAdmin() — zaštita admin stranica
  • login() — provjerava email/lozinku pomoću password_verify()
  • registerUser() — validacija i registracija sa password_hash()
  • sobaJeDostupna() — SQL upit koji provjerava preklapanje rezervacija u datom periodu
  • izracunajCijenu() — broj noćenja × cijena po noći

index.php — Pregled soba

Glavna stranica aplikacije. Tok izvršavanja:

  1. Učitava auth.php za sesiju i korisnika
  2. Čita GET parametre tip i pretraga za filtriranje
  3. Gradi dinamički SQL upit sa prepared statements (zaštita od SQL injection)
  4. Prikazuje sobe u grid karticama
$sql = 'SELECT * FROM sobe WHERE dostupna = 1';
if ($tip) { $sql .= ' AND tip = ?'; }
$stmt = getDB()->prepare($sql);
$stmt->execute($params);

rezervacija.php — Online rezervacija

Zaštićena stranica (requireLogin()). Pri POST zahtjevu:

  1. Validira datume (dolazak < odlazak, ne u prošlosti)
  2. Provjerava kapacitet sobe
  3. Poziva sobaJeDostupna() da spriječi duple rezervacije
  4. Računa cijenu i upisuje rezervaciju sa statusom na_cekanju

includes/header.php i footer.php

Zajednički HTML dio koji se uključuje na svakoj stranici pomoću require_once. Header prikazuje navigaciju, ime korisnika i flash poruke. Footer učitava main.js. Ovim se izbjegava ponavljanje istog koda na svakoj stranici (DRY princip).

setup.php

Jednokratna instalaciona skripta. Čita sql/schema.sql, kreira bazu i tabele, zatim dodaje test korisnike (admin i gost) sa sigurno haširanim lozinkama.

5. Frontend — HTML, CSS i JavaScript

assets/css/style.css

Glavni CSS fajl sa CSS varijablama za boje (:root). Sadrži stilove za:

  • Header i navigaciju
  • Kartice soba (grid layout, flex za poravnanje dugmadi)
  • Forme i dugmad
  • Tabele za admin panel
  • Responzivni dizajn (@media za mobilne uređaje)

assets/js/main.js

JavaScript se izvršava nakon učitavanja stranice (DOMContentLoaded):

  • initReservationForm() — dinamički izračun cijene rezervacije dok korisnik bira datume
  • initRegisterForm() — provjera da se lozinke poklapaju prije slanja
  • initDateValidation() — postavlja minimalni datum na današnji dan
const nocenja = Math.ceil(diffMs / (1000 * 60 * 60 * 24));
const ukupno = nocenja * cijenaPoNoci;

7. Tok podataka — primjer rezervacije

Korisnik otvori index.php → Klikne "Rezerviši" → login.php (ako nije prijavljen) → rezervacija.php → PHP validacija → INSERT u bazu → moje-rezervacije.php

Sigurnosne mjere u kodu

  • Prepared statements — svi SQL upiti koriste prepare() i execute() sa parametrima
  • password_hash / password_verify — sigurno čuvanje lozinki
  • htmlspecialchars (e()) — zaštita od XSS pri prikazu korisničkih podataka
  • Sesije — autentifikacija bez slanja lozinke pri svakom zahtjevu
  • requireLogin / requireAdmin — kontrola pristupa stranicama

8. Pokretanje aplikacije

  1. Pokreni MySQL: brew services start mysql
  2. Instaliraj bazu (samo prvi put): php setup.php
  3. Pokreni PHP server: 
    cd "/putanja/do/projekta"
php -S localhost:8000
  4. Otvori browser: http://localhost:8000
